El presente proyecto trata de un sistema de ambulancias en el cual se implementaron las siguientes funcionalidades:
Se posee una lista de llamadas y un hashmap de ambulancias, en el cual el usuario podra asignar la ambulancia mas
cercana en ese momento a la llamada en cuestion. Asi mismo el sistema posee una manera de mostrar la lista de las
llamadas en el momento, como tambien la forma de agregar nuevas llamadas al sistema y nuevas ambulancias.
La interface InterfazSistema posee 4 metodos asignados que corresponden a las funcionalidades que estaran disponibles al usuario


Durante el desarrollo del proyecto se dio cuenta de que ciertas funcionalidades o metodos eran innecesarios
para el desarrollo de este o estaban en el lugar equivocado, asi mismo, se eliminaron las funciones 
obtener coordenadas(porque era mas facil usar un getUbicacion) y se movio el metodo asignar ambulancia 
a SistemaEmergencias porque tenia mas sentido.
